HTML არის ენა, რომელიც ქმნის ვებ გვერდებს, მარტივი კოდების გამოსაყენებლად, როგორიცაა: სურათები, ტექსტები, ბმულები.

CSS არის ენა, რომლის საშვალებითაც ხდება საიტის სტილის შეცვლა. CSS-ის საშვალებით ხდება ფოტოების და ტექსტის განლაგება, რათა საიტი კარგად გამოიყურებოდეს.

JavaScript არის ენა, რომელიც საიტს ინტერაქიტიულს ხდის. JavaScript-ით შეგვიძლია ანიმაციები გავაკეთოთ, ღილაკს მოქმედების უნარი მივცეთ.
